## 🚀 核心功能
- **语义搜索**：直接输入自然语言需求（例如“找个做数字人的工具”），无需精准关键词。
- **5000+ 应用收录**：深度筛选全球优质 AI 应用。
- **保姆级教程**：为每一个工具提供详细的实操指南和难度、价格分析。
- **高性能架构**：采用向量检索与重排技术，极速响应推荐需求。

## 🛠 技术栈
- 前端：React / Next.js
- 后端：Node.js / Express
- AI 模型：DeepSeek-V3 / DashScope Rerank
- 数据库：MySQL + 向量相似度检索 (Vector Search)
